# Análisis de Ventas y Rentabilidad - Superstore

Proyecto de análisis de datos utilizando el dataset Superstore.
Incluye análisis exploratorio, visualización de métricas clave
y un modelo básico de Machine Learning (Regresión Lineal).

## Contenido del proyecto
- Dataset original
- Notebooks de análisis y Machine Learning
- Gráficos generados en Python
- Dashboard
- Presentación final
- Sheets con métricas resumidas

## Tecnologías utilizadas
- Python
- Pandas
- Matplotlib
- Scikit-learn
- Google Sheets
- Looker Studio
